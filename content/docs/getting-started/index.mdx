## Quick Start

<Steps>
  <Step>
    ### Environment Setup

    Follow the [Environment Setup Guide](/docs/getting-started/environment-setup) to configure your development environment with all required API keys and credentials.
  </Step>
  <Step>
    ### Database Setup

    Initialize your PostgreSQL database using the [Database Guide](/docs/getting-started/database-setup). The platform uses **Prisma** as the ORM with **Neon** serverless Postgres.

    ```bash title="Database setup"
    npx prisma migrate dev
    npx prisma db seed
    ```
  </Step>
  <Step>
    ### Install & Run

    <Tabs items={['npm', 'pnpm']}>
      <Tab value="npm">
        ```bash title="Terminal"
        npm install
        npm run dev
        ```
      </Tab>
      <Tab value="pnpm">
        ```bash title="Terminal"
        pnpm install
        pnpm dev
        ```
      </Tab>
    </Tabs>

    The application will be available at [http://localhost:3000](http://localhost:3000).
  </Step>
</Steps>
